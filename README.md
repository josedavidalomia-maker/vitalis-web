# Vitalis Empresarial® — Sitio web oficial

Sitio web de **Vitalis Empresarial®** — Consultoría en riesgo psicosocial, SST y bienestar organizacional. Cali, Colombia.

## Estructura
- `index.html` — Aplicación de página única (SPA): 14 secciones, formulario de contacto con envío automático, diagnóstico IVSO con registro de prospectos y política de datos (Ley 1581/2012).
- `img/` — Logo (WebP), favicons e imagen Open Graph.

## Publicación
El sitio es 100% estático: puede servirse desde GitHub Pages, o cualquier hosting. No requiere backend.

**Configuración pendiente tras el despliegue:**
1. Pegar la URL del Apps Script en `var SHEET_WEBAPP_URL = ''` (registro IVSO → Google Sheets).
2. Activar FormSubmit con un primer envío de prueba.
